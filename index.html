Testing fonts in SVG

This page shows options for adding fonts to SVG files. We want to find a method for adding fotns to SVG that gets us the font in the text on this page – Kalam – in the SVGs.

The SVGs here are linked to in img elements.

Font behaviour depends on whether the font is embedded in a web page or inline. The iframe beside each image shows the SVG as a standalone page. Or click on an image to see it on its own, and to see whether the font loads.
